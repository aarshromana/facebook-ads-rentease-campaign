# Facebook Ads Campaign – RentEase

## Overview
This project is a simulated Facebook Ads campaign for a furniture rental startup called RentEase.

## Objective
To generate leads and increase awareness using Meta Ads Manager.

## Target Audience
- Age: 22–35
- Location: Metro cities
- Interests: Furniture, home decor

## Strategy
- Lead Generation campaign
- Image, Video & Carousel ads
- Budget: ₹300–₹500/day

## Results (Simulated)
- CTR: 2.5%
- CPC: ₹8
- Leads: 120

## Tools Used
- Meta Ads Manager
- Canva / Gamma
